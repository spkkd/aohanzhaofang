# aohanzhaofang
本软件由 基于微信社交平台运行的敖汉找房微信小程序和基于浏览器运行的敖汉找房数据管理平台 组成, 数据管理平台建设的目的是为方便业务人员可以及时在平台内发布房源等必要信息到微信小程序平台,让使用者可以及时看到,业务人员可以随时在数据管理平台观测到数据的变化并更加方便的进行数据的管理。

### 平台简介

   本软件由 基于微信社交平台运行的敖汉找房微信小程序和基于浏览器运行的敖汉找房数据管理平台 组成, 数据管理平台建设的目的是为方便业务人员可以及时在平台内发布房源等必要信息到微信小程序平台,让使用者可以及时看到,业务人员可以随时在数据管理平台观测到数据的变化并更加方便的进行数据的管理。
